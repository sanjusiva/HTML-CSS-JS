EVENT REGISTRATION FORM










     


Requirements

  • Create an user interface for Event Registration Form.
  • This page should contain first name, last name, company name, email id, phone no and subject.
  • All the fields should be validated. First name, last name, company name, subject should accept only alphabets.
  • Phone number should accept only 10 digit numbers.
  • Email Id should be in the following format example@email.com
  • Implement the above mentioned requirements by using html, css and javascript.
